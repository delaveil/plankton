# plankton
Здесь собраны мои основные учебные проекты (схемы баз данных и виртуальная машина) и наработки в разных языках программирования (с++ и python).

📁 Структура репозитория

- ** c++ **  
  Решения задач и лабораторные работы с курса «Введение в программирование (C++)» — Академия Яндекса / Stepik.

- ** python **  
  Jupyter-ноутбуки, скрипты анализа данных.

- ** базы данных + Ubuntu **  
  Полноценные схемы базы данных для бизнеса (реляционная модель + EER-диаграмма) + настройка виртуальной машины Ubuntu с работой в терминале.
  
Внутри каждой папки — готовые проекты, диаграммы, код и скриншоты. Можно смотреть прямо в браузере.
